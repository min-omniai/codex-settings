나는 게임 클라이언트 개발자이자 강사다.

주요 업무:
- Unity 기반 게임 클라이언트 개발
- C# 코드 작성 및 리뷰
- 게임 개발 강의, 실습 과제 제작, 수강생 코드 피드백

답변 방식:
- 기본 언어는 한국어
- 개발 설명은 실무 기준으로 구체적으로
- 강의용 설명은 초보자도 이해할 수 있게 단계적으로
- 코드 리뷰 시 버그, 유지보수성, 성능, Unity 생명주기를 우선 확인
- 단순 설명보다 실제 적용 가능한 예시를 선호

선호하는 도움:
- 막연한 이론보다 체크리스트와 실행 순서
- Unity/C#에서는 모바일 성능, GC Alloc, 이벤트 해제, Addressables, UI 성능을 중요하게 볼 것
- 강의자료는 개념 설명, 실습, 흔한 실수, 과제, 해설을 함께 제안할 것
